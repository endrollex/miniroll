<span class="m7">minicode - easy editing XHTML</span><!--m7--><br />
<br />
<span class="m6" style="color: green" dir="ltr">Introduction:</span><!--m6--><br />
<br />
minicode is a lightweight markup language for <a class="m4b" target="_blank" href="https://github.com/endrollex/miniroll" style="max-height: 9003px;">miniroll</a><!--m4--> easy editing, like BBCode.<br />
The available tags are usually indicated by square brackets ([ ]) surrounding a keyword.<br />
It is designed for for Blog, thus let through unsafe HTML like &lt;script&gt;.<br />
minicode allow user to write HTML freely and unlimitedly.<br />
<br />
<span class="m6" style="color: green" dir="ltr">Syntax:</span><!--m6--><br />
<!--m5--><ol><li>The minicode cannot nest:
<pre style="background-color: #D3D3D3;">[url]http://goo.gl/[/url] OK
[b][url]http://goo.gl/[/url][/b] This expression is ERROR!!!
</pre>Because nest will be complex and bloated.
In this situation, you can use HTML directly.</li></ol><!--m5b--><span class="m6" style="color: green" dir="ltr">Keywords and Usage:</span><!--m6--><br />
<!--m5--><ol><li>
<pre style="background-color: #D3D3D3;">[code]echo ipconfig[/code]</pre>

<!--m5--></li></ol><!--m5b--><span class="m6" style="color: green" dir="ltr">Mechanism:</span><!--m6--><br />
<br />
Mechanism of minicode is base on &quot;Search and Replace&quot;, the process is reversible and very simply.<br />
It abbreviated commonly-used HTML tags to short keywords, the article can mix HTML.<br />
The result article is a HyperText of HTML tags, some with a little comment markup (&lt;!--...--&gt;) to reversion purpose.<br />
Some keywords of minicode depends CSS, the <a class="m4b" target="_blank" href="https://github.com/endrollex/miniroll/blob/master/htmindex.css" style="max-height: 9003px;">default CSS</a><!--m4--> provided to show the predefined style.<br />
